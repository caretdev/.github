# CaretDev

**Modern developer ecosystem for InterSystems IRIS**

Use InterSystems IRIS with modern tools and workflows:

* **Python** → SQLAlchemy, Django
* **TypeScript / Node.js** → TypeORM
* **Analytics** → dbt
* **AI / LLM** → LangChain, vector search
* **DevOps** → Testcontainers, CI/CD

---

## Why

InterSystems IRIS is powerful, but lacks integration with the modern developer ecosystem.

CaretDev builds the **missing layer** — bringing IRIS into Python, TypeScript, analytics, and AI workflows used by modern engineering teams.
