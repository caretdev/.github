# CaretDev

Modern developer ecosystem for InterSystems IRIS

## Use IRIS with modern tools

- Python → SQLAlchemy, Django
- TypeScript → TypeORM
- Analytics → dbt
- AI/LLM → LangChain, vector search
- DevOps → Testcontainers, CI/CD

## Why

IRIS lacks integration with modern developer ecosystems.

We build the missing layer.
